new UPOS-ULABEL treebank versions is here: /Users/gune00/data/MLDP/universal_treebanks_v2.0/

create UPOS tagger models:
- SvmTagger code and corpora in /Users/gune00/dfki/TIM-Tools/SvenTagger

- Steps: example EN
	- copy CONLL train/devel/test files to /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/UPOS_Corpora
	
	- transform CONLL to SVM corpus format: 
		awk -f ../corpusTools/extract.awk en-universal-test.conll > en-test-utag.svm
		
	- train models: cd /Users/gune00/dfki/TIM-Tools/SvenTagger/SVMTool/bin
		perl SVMTlearn config.short-upos-english -v 2
		
	- copy model files to /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/models/UPOS/english/
	- copy model files to /Users/gune00/dfki/workspaceKepler/MDP/resources/tagger/UposModels/english
	
	- create SVM test file: cd /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/UPOS_Corpora
		awk -f ../corpusTools/extract-wf.awk en-test-utag.svm > en-test-utag.svm.in
		
	- testing and evaluating model files: cd /Users/gune00/dfki/TIM-Tools/SvenTagger/SVMTool/bin
		./SVMTagger -V 2 -S LRL -T 0 /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/Models/UPOS/english/ENGLISH < /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/UPOS_Corpora/en-test-utag.svm.in  > /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/UPOS_Corpora/en-test-utag.svm.out	
		./SVMTeval 0 /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/Models/UPOS/english/ENGLISH  /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/UPOS_Corpora/en-test-utag.svm /Users/gune00/dfki/TIM-Tools/SvenTagger/corpora/UPOS_Corpora/en-test-utag.svm.out
